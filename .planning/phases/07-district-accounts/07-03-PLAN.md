   - `"use server"` directive
   - Export `getDistrictBySlug(slug: string)` -- query districts table WHERE slug = param AND verified = true
   - Returns district info (id, name, website, verified_at) + count of open jobs
   - Export `getDistrictJobs(districtId: string)` -- query jobs WHERE claimed_by_district_id = districtId AND is_active = true AND delisted_at IS NULL, join schools for school name
   - Returns array of jobs with id, title, school name, city, school_type, first_seen_at, url

2. Create `src/lib/queries/get-all-districts.ts`:
   - `"use server"` directive
   - Export `getAllVerifiedDistricts()` -- query districts WHERE verified = true, ORDER BY name ASC
   - Returns array of { id, name, slug, website, jobCount } where jobCount is a subquery counting active claimed jobs
