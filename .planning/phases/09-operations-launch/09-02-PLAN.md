1. Create `src/app/sitemap.ts`:
   - Export default async function returning MetadataRoute.Sitemap
   - Query active jobs (id, updated_at) from jobs table using createAdminClient
   - Build sitemap entries:
     - Static pages: / (priority 1, daily), /jobs (priority 0.9, daily), /about (priority 0.5, monthly), /coaching (priority 0.5, monthly)
     - Dynamic: one entry per active job at /jobs/{id} (priority 0.8, daily, lastModified from updated_at)
   - Base URL: "https://school-job-portal.vercel.app"

2. Update `src/app/jobs/[id]/page.tsx`:
   - Add JSON-LD structured data as a script tag in the page component:
     ```
     const jsonLd = {
       "@context": "https://schema.org",
       "@type": "JobPosting",
       title: job.title,
       description: job.description || "",
       datePosted: job.first_seen_at,
       hiringOrganization: {
         "@type": "Organization",
         name: job.schools?.name || "Unknown School",
       },
       jobLocation: {
         "@type": "Place",
         address: {
           "@type": "PostalAddress",
           addressRegion: "PA",
           addressLocality: job.schools?.city || undefined,
         },
       },
     };
     ```
   - Render: `<script type="application/ld+json" dangerouslySetInnerHTML={{ __html: JSON.stringify(jsonLd).replace(/</g, "\\u003c") }} />`
   - XSS protection: the .replace() prevents script tag injection from scraped descriptions
   - Enhance generateMetadata to include OG tags:
     - openGraph: { title, description, type: "website", url: `https://school-job-portal.vercel.app/jobs/${id}` }
     - If salary info exists in the job, mention it in description
