**Create `src/lib/actions/coaching-action.ts`** — server action:
- "use server" directive
- Import Resend, instantiate with `process.env.RESEND_API_KEY`
- Export `submitCoachingRequest(formData: FormData)` function
- Extract fields: name (required), email (required), currentRole (optional), yearsExperience (optional), positionSought (optional), message (optional), phone (optional)
- Validate name and email are non-empty; return `{ success: false, message: "Name and email are required" }` if missing
- Send email via `resend.emails.send()`:
  - from: `"PA Educator Jobs <onboarding@resend.dev>"` (Resend test sender — no domain verification needed for MVP)
  - to: `[process.env.OPERATOR_EMAIL || ""]`
  - subject: `"Coaching Request from {name}"`
  - replyTo: submitter's email
  - html: simple formatted HTML showing all submitted fields (name, email, role, experience, position, message, phone)
- Return `{ success: true, message: "Thanks! I'll be in touch within 48 hours." }`
- If Resend errors or RESEND_API_KEY is missing, return `{ success: false, message: "Unable to send your request right now. Please try again later." }`
- Do NOT instantiate Resend at module level — do it inside the function to avoid errors when API key is missing
