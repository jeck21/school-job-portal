### Integration Issue (Low Severity)

**Unspecified toggle active-filter counter mismatch:**
- Desktop (`search-filter-bar.tsx:43`): `filters.unspecified !== true` — counts toggle as active when OFF
- Mobile (`mobile-filter-drawer.tsx:32`): `filters.unspecified !== false` — counts toggle as active when ON
- **Impact:** Active-filter badge count displays differently on mobile vs desktop for the same toggle state. Actual RPC filtering behavior is correct in both — only the display indicator is asymmetric.
- **Affected requirements:** UI-02, UI-04
